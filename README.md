# 🌱 OrtoManager v2.0 - AGGIORNATO

**OrtoManager** è un'app completa per gestire il tuo orto e giardino. Funziona completamente offline grazie alla tecnologia PWA!

## ✨ NOVITÀ VERSIONE 2.0

### 🆕 Funzionalità Aggiunte

1. **Database Piante Predefinite** 📚
   - 10 piante precaricate
   - Informazioni complete
   - Selezione rapida

2. **Campo "Numero Piante/Semi"** 🔢
   - Traccia quantità piantata
   - Visualizzato su ogni pianta

3. **Pagina Impostazioni (ALTRO)** ⚙️
   - Geolocalizzazione GPS 📍
   - Salvataggio città automatico
   - Gestione Notifiche 🔔

4. **Dialog Aggiungi Pianta** ➕
   - Selezione da database
   - Suggerimenti automatici
   - Form migliorato

## 🚀 Installazione

```bash
npm install
npm run dev
npm run build
```

Vedi INSTALLAZIONE.md per dettagli completi.

## 📱 Installa sul Telefono

1. Deploy su Vercel/Netlify
2. Apri link sul telefono
3. Aggiungi a Home
