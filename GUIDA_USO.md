# 🌱 Guida Rapida - OrtoManager

## Benvenuto in OrtoManager! 👋

OrtoManager è la tua app personale per gestire orto e giardino. Ecco come usarla al meglio!

## 📱 Primi Passi

### 1. Esplora l'App
L'app è divisa in 4 sezioni principali, accessibili dalla barra in basso:

- 🏠 **Home**: Dashboard con panoramica generale
- 🌱 **Piante**: Catalogo delle tue piante
- 📅 **Calendario**: Task e attività
- 👨‍🍳 **Ricette**: Ricette con i tuoi prodotti

### 2. Dati di Esempio
Al primo avvio troverai già alcuni dati di esempio:
- 5 piante (pomodori, basilico, zucchine, fragole, rosmarino)
- Alcuni task da completare
- 3 ricette italiane

Puoi usarli per familiarizzare con l'app, poi eliminarli quando vuoi.

## 🌿 Gestione Piante

### Aggiungere una Pianta
1. Vai su **Piante** (icona foglia)
2. Tocca il pulsante **+** in alto a destra
3. Compila i dettagli:
   - Nome della pianta
   - Categoria (ortaggio, frutto, erba, fiore, albero)
   - Data di semina/trapianto
   - Posizione nell'orto
   - Frequenza di irrigazione
   - Note personalizzate

### Annaffiare le Piante
- Ogni pianta mostra se ha bisogno di acqua
- Tocca il pulsante **"Annaffia"** per registrare l'irrigazione
- Il sistema terrà traccia automaticamente dell'ultima volta

### Filtrare le Piante
- Usa la **barra di ricerca** per trovare piante specifiche
- Tocca le **categorie** per filtrare per tipo
- Visualizza solo le piante che ti interessano

## 📆 Gestione Calendario

### Creare un Task
1. Vai su **Calendario** (icona calendario)
2. Tocca il pulsante **+**
3. Inserisci:
   - Titolo del task
   - Descrizione (opzionale)
   - Data di scadenza
   - Priorità (bassa, media, alta)
   - Categoria (irrigazione, concimazione, potatura, ecc.)

### Completare un Task
- Tocca il **cerchio** a sinistra del task
- Il task verrà segnato come completato ✅
- Puoi visualizzare i task completati cambiando il filtro

### Visualizzazione Task
I task sono raggruppati automaticamente:
- **Scaduti** (in rosso, da completare urgentemente)
- **Oggi** (da fare oggi)
- **Domani** (per pianificare)
- **Questa settimana**
- **Questo mese**
- **Prossimamente**

## 👨‍🍳 Ricette

### Trovare Ricette
- Usa la **barra di ricerca** per cercare ricette
- Filtra per **difficoltà** (facile, media, difficile)
- Filtra per **stagione**
- Visualizza solo i **preferiti**

### Aggiungere ai Preferiti
- Tocca l'icona del ❤️ sulla ricetta
- Le ricette preferite sono evidenziate
- Usa il filtro "Preferiti" per vederle tutte insieme

### Informazioni Ricetta
Ogni ricetta include:
- ⏱️ Tempo totale di preparazione
- 👥 Numero di porzioni
- 🏆 Livello di difficoltà
- 🌿 Ingredienti necessari
- 📝 Istruzioni passo-passo
- 🏷️ Tag per categorizzare

## 💡 Consigli Utili

### Backup dei Dati
I tuoi dati sono salvati localmente sul telefono. Considera di:
- Fare screenshot importanti
- Esportare periodicamente (funzione in arrivo)
- Non disinstallare l'app se non vuoi perdere i dati

### Notifiche
- Abilita le notifiche per ricevere promemoria
- Sarai avvisato dei task in scadenza
- Riceverai suggerimenti per l'irrigazione

### Uso Offline
- L'app funziona **completamente offline**
- Tutti i dati sono sul tuo dispositivo
- Non serve connessione internet dopo l'installazione

### Organizzazione
Per un orto ben organizzato:

1. **Aggiungi tutte le tue piante** con dettagli completi
2. **Usa le note** per informazioni specifiche
3. **Imposta la frequenza di irrigazione** corretta
4. **Crea task ricorrenti** per lavori regolari
5. **Aggiungi ricette** quando raccolgi prodotti

## 🎯 Funzioni Avanzate

### Note Personalizzate
- Aggiungi note a ogni pianta
- Registra osservazioni importanti
- Traccia problemi o successi

### Task Collegati alle Piante
- Collega i task a piante specifiche
- Tieni traccia di tutte le cure per ogni pianta
- Visualizza lo storico delle attività

### Ricette Stagionali
- Le ricette sono filtrabili per stagione
- Usa i prodotti del tuo orto al momento giusto
- Pianifica le semine in base alle ricette preferite

## ❓ Domande Frequenti

**Q: I dati sono sincronizzati sul cloud?**
A: No, tutti i dati sono salvati localmente sul tuo dispositivo per massima privacy e funzionamento offline.

**Q: Posso usare l'app su più dispositivi?**
A: Al momento no, i dati sono locali. Una funzione di esportazione/importazione è in programma.

**Q: L'app funziona senza internet?**
A: Sì! Dopo l'installazione iniziale, funziona completamente offline.

**Q: Come faccio il backup?**
A: Per ora puoi fare screenshot. Una funzione di esportazione dati è in sviluppo.

**Q: Posso personalizzare le categorie?**
A: Le categorie sono predefinite per semplicità. Usa le note per informazioni personalizzate.
